INSTALL MANUAL
===============================================================================

Requirements
-------------------------------------------------------------------------------
vim 

Install
-------------------------------------------------------------------------------

Get lazyvim
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
download lazyvim source code or checkout Git Repo

Install lazyvim
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
..

    $ cd path/to/lazyvim

    $ chmod a+x install.sh

    $ ./install.sh

Backup
-------------------------------------------------------------------------------
Script named install.sh will backup your older user configration file ($HOME/.vimrc) to $HOME/.vimrc.old automatically.
